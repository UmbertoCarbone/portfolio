# Portfolio

Progetto full-stack per il portfolio di Umberto.

---

## Backend

### Tecnologie

- **Node.js**: Runtime JavaScript lato server
- **TypeScript**: Superset di JavaScript con tipizzazione statica
- **Express**: Framework per la creazione di API REST

### Dipendenze

**Produzione:**

- `express` (^5.2.1): Framework per gestire route e richieste HTTP
- `dotenv` (^17.3.1): Caricamento variabili d'ambiente da file `.env`
